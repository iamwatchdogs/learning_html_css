Font-family and External font

Font family is more of the most important css styling used on the text content of an html page.
Font family is an property which focus on the styling of the text content.
We can change the font of an text just by givin values like san-serif, Arial, Helvetica to name a few, to the selector font-family.
For example, 

            .change-font-to-Helvetica {
                color: aqua;
                font-family: Helvetica;
            }

we can also stack up multiple fonts, just to make sure that If one fails, the alternate one will takeover.
The font loading problems can be caused when the specific font is not present in the system where this html page is viewed from.
To illustrate an example of stacking up multiple fonts, 

            .change-font-to-the-following {
                color: lightcoral;
                font-family: "Franklin Gothic Medium", "Arial Narrow", Arial, sans-serif;
            }

Adding External Font

As we have discussed above about the font-failure problem, There is a way to add the required font even if the system does not have it.
And this is possible by add an external font from online servers which contains multiple fonts. One of the most popular choice to add external fonts is throught Google fonts.
All we have to do is to select an font on Google Font, After selecting the font ( you can customize if you want ),
the selected font will provide an html linking tag by setting the hyperlink to the location contain the specified font.
We have to copy it and paste it in the head section right above where you're going to use the font-style in either external CSS or even Internal CSS.
Now, we can use the font despite the system being used.
Even throught it safe to use the external font directly, it's recommended to stack it with some alternative just to avoid the worst case of displaying,
the sam default font for all the text content throughtout the html page.

Now, lets see an example, 

            .change-font-to-external-font-light-ubuntu {
                color: lightgreen;
                font-family: 'Ubuntu', sans-serif;
            }
