FlexBox

FlexBox is one of the crucial features of the CSS which provides the ease of alignment of elements just by specifying the flexbox properties in parent and its child elements.

We can use this feature only when we have set the display property as flex

FlexBox - Parent properties :

1) display

As we have discussed earlier, we have to keep the value of display property as flex to use the flexbox features.

.container { display: flex; }

2) flex-direction

flex-direction is used to set the display of document flow of the children of the specified container. Its values can be row, row-reverse, column, column-reverse, ( where the row is the default value ).

for example,

.flex-direction-column-reverse { flex-direction: column-reverse; }

3) flex-wrap

flex-wrap property is used to wrap the children elements while resizing the browser / display window. This property encourages responsive-ness. This property can have the following values; wrap, wrap-reverse, nowrap (default).

Note:
Note that giving a relative sizing units to the children element will not show any affect of these properties as they are already responsive and will resizing themselves to adjust to the new size.

for example,

.flex-wrap-wrap { flex-wrap: wrap; }

4) justify-content

FlexBox does alignment with help of two asix i.e., the main axis ( ~ x-axis ) and the cross axis ( ~ y-axis ). By specifying justify-content property, we can define the alignment across the main axis.

for example,

.justify-content-center { justify-content: center; }

5) align-items

FlexBox does alignment with help of two asix i.e., the main axis ( ~ x-axis ) and the cross axis ( ~ y-axis ). By specifying align-items property, we can define the alignment across the cross axis.

for example,

.align-items-center { align-items: center; }

6) align-content

while we are using flex-wrap: wrap;, we can see that while wrapping the content according to the new size ( while resizing ) there will some additional space between the content. To remover the additional space, we can use this property,

for example,

FlexBox - Child properties :

1) Order

order property is used in child whose parent has property display: flex. This property in which order the children elements should be displayed.

for example,

.order-3{ order: 3; }
.order-1{ order: 1; }
