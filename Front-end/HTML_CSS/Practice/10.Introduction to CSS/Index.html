Introduction to CSS


CSS means Cascading Style Sheets. These are used to provide moden web page level styling to a basic html page.

There are basically three types of CSS implentations and they are:

  • Inline-css
  • Internal-css
  • External-css

Inline CSS:

Inline CSS is applied by specifing the styling of the content by using the style attribute. Just like this paragraph is in green color

Internal CSS:

Internal CSS is applied by specifing the styling for the selectors in the head of the html page using Sytle tag. Just like this paragraph is in red color.

External CSS:

External CSS can be implented by linking and external css file using the link attribute in the head section of the html page. Just like this paragraph is in blue color.
