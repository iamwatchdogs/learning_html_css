Tables :-

We can implement the table in the html by using the following tags :

  • Table tag ( <table> )
  • Table Heading ( <th> )
  • Table Row tag ( <tr> )
  • Table Data tag ( <td> )

We can also add semantics to tables, by using the following tags :

  • caption ( <caption> )
  • Table Header ( <thead> )
  • Table body ( <tbody> )
  • Table Footer ( <tfoot> )
