Adding Images


We can add an image using the tag <imag>. This tag does not have a closing tag.
We add a '/' at the end of the image tag, it mean the tag is self closing.
It really matters when we are using React. but from HTML5 it really doesn't matter

The image tag consists of mainly 2 types of attributes and they are: "src" and "alt" attributes.
The 'src' attribute is used to specify the source of the image tag and,
the 'alt' attribute is used to specify the alternative description of the image tag which means,
when the src attribute is broken then it will display the alt attribute value instead.
If you want you can also add title to the image and it works just like the abbreviation tag for images.

Syntax: <imag src="location" alt="text title="text" />

We can add simply add images to the page using the above info.
But, Now a days it has become crucial to add/specify few more elements.
And they are:
"width", "height", "loading"

widith - specifies the width of the image.
height - specifies the height of the image
loading - specifies the mode of loading the image.

We need to specify the width and height of the image to avoid problems caused by cummulative layout shift,
this happens when the html page is loaded in a different device than the current browser.
By specifying the width and height, we can tell the html page to leave a certain part of the page from the image.

by default, the loading attribute is set with "eager" mode which mean it will load the image at the time of loading
the page immediately, This may delay loading of the html page. By setting few images with "lazy" mode of loading,
it helps the page load even faster. The lazy mode can be useful when we assgin the mode to the image which below the
fold ( fold means the part of the page we are able to see ).

Syntax: <imag src="location" alt="text title="text" widith="num px" height="num px" loading="mode" />

to show case the lazy mode we have to create a lot of space to create a lower fold.
So to create them we will type "br*num_of_times" to create a lower fold.
