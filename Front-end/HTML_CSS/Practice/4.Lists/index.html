List :-

Ordered Lists :-

Ordered list are the list in which each individual list item ( <li> ) is denoted using (by default) numbers,
Just as given below,

  1. First we have to use attribute <ol> to create a list.
  2. Then we can add as many list items as we want using <li> attribute.
  3. Then it will display the list as seperate point/items denoted by numbers.

In Addition to that, we can also change the types in ordered as roman numbers, alphabets, etc...

For example :

  1. First we have to use attribute <ol> to create a list.
  2. Then we have to specify the type as ( <ol type = 'a'>, for this example ).
  3. Then we can add as many list items as we want using <li> attribute.
  4. Then it will display the list as seperate point/items denoted by numbers.
Unordered Lists :-

Unordered list are the list in which each individual list item ( <li> ) is denoted using (by default) dots,
Just as given below,

  • It is similar to the Ordered list but the only different is that, Ordered list uses numbers and Unordered don't.
  • So, First we have to use attribute <ul> to create a list.
  • Then we can add as many list items as we want using <li> attribute.
  • Then it will display the list as seperate point/items denoted by numbers.
Description List :-

Description List is a trick one. Unlike other two list attribute, Description list has two attribute per item
and they are used just as given below,

Step One :
We create a description list using <dl> attribute.
Step Two :
we create a description title using <dt> attribute within the description list.
Step Three :
After giving a description title, we will give description detail to the title using <dd> attribute under the description title.
Step Four :
It's done, the output will be shown in a sub heading hierarchy.
Note :
Most of this html is done using description list.
