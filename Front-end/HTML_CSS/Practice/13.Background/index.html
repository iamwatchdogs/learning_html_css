Background Property:-

Background Properties are one of the important properties of css.

Some of the background properties are:

1) Background Color:

Color is one of the basic properties of css. As it says, it add the color to the background of the elements of the selector. for example, 

                        .steel-blue{
                            background-color: steelblue;
                        }

2) Background Image:

We can add images to background images using background-image Property and for the value, we can specify it using url(). For example, 

                    .example-bgimage{
                        background-image: url(../src/img/caribbean_400_x_225.jpg);
                    }

3) Background Repeat:

As we can see the above image, it's repeated to fit the proportion of the div element of class example-bgimage. By default, the background-repeat Property has the value of repeat. The other value we have is no-repeat.

No Repeat value will stop the image from repeating itself to fit the selector size. No-Repeat will display the image only once and with its original size/proportions. For example, 

                    .example-bg-no-repeat-image{
                        background-color: steelblue;
                        background-image: url("../src/img/caribbean_400_x_225.jpg");
                        background-repeat: no-repeat;
                    }

4) Size:

By using background-size property, we can display the image with given size. The value for this property can be given in form of width height (or) predefined value like - cover, contain. 

                example 1 :
                
                    .example-bg-img-size{
                        background-color: steelblue;
                        background-image: url("../src/img/caribbean_400_x_225.jpg");
                        background-size: 40% 70%;
                    }
                
                example 2 :

                    .example-bg-img-size-cover{
                        background-color: steelblue;
                        background-image: url("../src/img/caribbean_400_x_225.jpg");
                        background-size: cover;
                    }
                
                example 3 :

                    .example-bg-img-size-contain{
                        background-color: steelblue;
                        background-image: url("../src/img/caribbean_400_x_225.jpg");
                        background-size: contain;
                    }

Gradient

Gradient displays the transition between different colors. Gradients can be applied in two ways i.e., the linear-gradient way or radial-gradient way.

I) Linear Gradient way

We can apply linear gradient to background assigning linear-gradient( to direction, color1, color2, ... ). For example, 

                    example 1:
                        .example-linear-gradient-direction{
                            background: linear-gradient( to right bottom, #0000ff, steelblue, rgba(70,130,180,0.65))
                        }
                    
                    example 2:
                    .example-linear-gradient-deg{
                        background: linear-gradient( 260deg, #0000ff, steelblue, rgba(70,130,180,0.65))
                    }

II) Radial Gradient way

We can apply radial gradient to background assigning radial-gradient( shape (optional), color1 amount1, color2 amount2, ... ). For example, 

                    example 1:
                        .example-radial-gradient{
                            background: radial-gradient( blue, steelblue);
                        }
                    
                    example 2:
                        .example-radial-gradient-with-shape{
                            background: radial-gradient( circle, blue 25%, steelblue 75%);
                        }
