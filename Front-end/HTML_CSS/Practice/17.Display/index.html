Display Property

All of the elements in html is divided into two types i.e., Block elements and inline elements.

The inline elements are the elements which take the only space which is required to fit its contents.

The Block elements are the elements are the elements which take up whole block-sized spacing for its content. These elements don't let other element around its boundary, like an introvert...

While setting up different elements in html page, we might have a suitation where we need to set element in the same line. And this might be problematic when using block elements. So, we use the Display property.

Now, Let's see some of the values that can be assigned to this property and thier examples :

1) Inline

By assigning this value to display property, we can change element into an inline element i.e., when we change an block element into inline element then irrespective of its width and height ( even mentioned ones ) the element will be occupying only the required space.

3) Inline-Block

In some suitation, we might want an element to be an block element, consisting its contents within its boundaries, and also show similar properties to that of the inline block. In such suitation, we use inline-block property value.

The elements assigned with display property with inline-block value works like an hybrid of inline elements and block elements. These elements are like the block elements which support inline property (or) the block elements which allow other elements to be placed side-ways of this element as it won't be occupying the extra space like an original block element.
