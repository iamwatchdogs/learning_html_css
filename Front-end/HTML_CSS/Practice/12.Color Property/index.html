Color properties :-

color property for a selector can be applied by using the property called color:. Most of the time it referes to the text content within the specified selectors. we can also apply it to background by using the property called background-color: or background:.

The value to the color property can be of the following types:

1) Color Name:

We can directly specify value to the color property just by giving the respective CSS color name. For example:

                    .steel-blue {
                        color: steelblue;
                    }

2) Hex code:

hex code is a value which represents a color in hexcode format. basically it's format in the form of : #RRGGBB which means that first 2 hex digits represents the amount of red color, the next 2 hex digits represents the amount of green color, and last 2 represents the amount of blue color. For example: 

                    .green {
                        color: #00ff00;
                    }

3) RGB parameters:

Here we provide the amount of red, green, blue directly as the parameters of rgb() ranging from 0 to 255. For example:

                    .red {
                        color: rgb(255, 0, 0);
                    }

4) HSL parameters:

It is quite Similar to rbg() value. But here we provide the amount of Hue, Saturation, Lightness as the parameters of hsl() where the h value will be ranging from 0 to 255 and the s, l values ranging from 0 to 1 (or) 0% to 100%. For example: 

                    .light-blue {
                        color: hsl(180, 100%, 50%);
                    }

5) RGBA parameters:

This parameterized value is just an extension of rgb(). The additional value included i.e., 'a' is the alpha which decides the opacity/transparence of the color. The value of a ranges from 0 to 1 (or) 0% to 100% For example: 

                    .tranparent-red {
                        color: rgba(255, 0, 0, 50%);
                    }

6) HSLA parameters:

This parameterized value is just an extension of hsl(). The additional value included i.e., 'a' is the alpha which decides the opacity/transparence of the color. The value of a ranges from 0 to 1 (or) 0% to 100% For example: 

                    .tranparent-light-blue {
                        color: hsla(180, 100%, 50%, 50%);
                    }
