Text & Basic Font Properties

In this practice page, we will be discuss about few Text Manipulation & Some Basic Font Properties of CSS.
And also, we will not be discussing about all types of value used for the properties because it would be time comsuming.

1) Text-Alignment

Text alignment works similar to text alignment present in MS office applications. I have set up this page such way that all the text element are centered. The text alignment can have other value like left, right, justify. For example, 

                    
                        .align-to-justify {
                            text-align: justify;
                        }
                    
                    
                         Note that the content is specify within the
                        <pre> tag which has indentation.

2) Text-Decoration

Text decoration is used to decorate the text of an element by values such as dashed, dotted, double, e.t.c. We can also remove the default decoration by giving the value none. Let's see one of this property:

.decorate-overline { text-decoration: overline; }


3) Text-Transfrom

Text transfrom property is used to transfrom the state of text into varios from like captilize, uppercase, lowercase and even none. Let's see one of this property: 

                    
                        .transfrom-text-to-uppercase{
                            text-transform: uppercase;
                        }

Till Now, we have seen only Text Manipulation properties; Now let's take a look at some basic Font properties.

1) Font-Size

As the name suggests, this font property deals with the sizing of font of the selector. For example, 

                    
                        .changed-font-size {
                            font-size: 1.5em;
                        }

2) Font-Weight

The font weight gives the boldness of the text of the selector. The value of this property ranges from 0 to 900 and mostly used in terms of multiples of 100.For example, 

                    
                        .changed-font-weight {
                            font-weight: 900;
                        }

3) Font-Style

The font-style is used to give basic styling for the text of the selectors. Well, mostly it's used for setting text in italic styles as shown below, 

                    
                        .changed-font-style {
                            font-style: italic;
                        }
