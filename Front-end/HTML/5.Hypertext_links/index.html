Introduction :-

  • Hyperlink is a way to link various files, section via various reference.
  • This can be done using anchor tag ( <a> ).
  • Anchor tag contain various attributes but one of the most common and important attribute is the "href" attribute.
  • "href" ( syntax: <a href="location/reference" >text</a> ) is the attribute which specifies the reference of the hyperlink.
  • Using Hyperlink we can redirect the current page into another page, move a particular section of the page, download, etc.
  • The hyperlink which you have visited will be in purple color by default.
  • The hyperlink which you haven't visited yet will be in slightly dark blue in color by default.
Note :
The color of the visited or not visited hyperlinks can be modified using "CSS".
That's why the colour of the hyperlink in this html page differs from the default ones

What is reference? :-

  • Reference ( "href" attribute ) is the source of the destination in the ancho tag.
  • Reference can either be absolute reference or relative reference.
Absolute Reference :
  • Absolute Reference is the type of reference which gives full exact location of the source.
  • The can it be refering to a source file present in a specific directory of our local machine; Or,
  • It can be a refering a particular website url.
example:
  • Click Google to open it in the same tab.
  • Click Google to open it in the new tab.
Relative Reference :
  • Relative Reference is the type of reference we use when the exact full reference location of the source file is not required.
  • This reference can be applied when the source file is present in the current directory of the html page.
  • It can refer to the local source files or the internal sections of the html page.
example:
  • Click About me to open it in the same tab.
  • Click About me to open it in the new tab.
Note :
By using any of the references you can create a hyperlink which not only redirect into other pages but also provide a option for creating a download link.
You can create a download link by using "download" attribute in anchor tag, just as given below,

  • Using Relative Reference : Download icon.

What are Sections? :-

  • Sections are the partition done to seperation into different part of the html page.
  • Sections helps us to easily navigate throught the html page with huge content.
  • We can divide out html page into section by using the tag "<section>".
  • If we use the sections tag then we must specific an attribute called "id".
  • without the attribute "id" we can not specify the reference to navigate throught the page.
  • Syntax: <section id="name_of_the_section"> content </section>
  • After creating sections in out html page, we have to use the anchor tag to navigate throught the page.
  • Syntax: <a href="#name_of_the_section"> hyper link statement </a>
